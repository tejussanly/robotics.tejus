What I Am Working On

Project Overview

A smart digital lock system built on Arduino that combines security, simplicity, and automation into one compact embedded project.

The Idea

I am building a password-protected digital lock using an Arduino UNO microcontroller. The system allows users to enter a 6-digit PIN via a 4×4 matrix keypad. If the correct password is entered, a servo motor rotates to unlock the door and automatically re-locks after a countdown.

Core Focus

The main focus is on security and user feedback. A 16×2 LCD display provides real-time prompts and status messages. A buzzer gives audio feedback on key presses, successful access, and failed attempts. After 3 wrong attempts, the system locks itself and triggers a lockout alert.

Key Capability

Users can also change the password at any time by entering the master PIN followed by a new 6-digit password, which is saved to EEPROM — so the custom password persists even after power loss. The default master password is 123456.

Simulation

The complete working simulation is hosted on Wokwi — an online Arduino simulator — so the project can be tested, inspected, and demonstrated without any physical hardware. The simulation is always accessible via the link in this documentation.

How the System Works

User presses keys on the 4×4 keypad. Each digit is masked as * on the LCD. Buzzer beeps on each keypress for tactile feedback.

Pressing # submits the entered password. Arduino compares it against the stored master password in EEPROM.

After 10 seconds in the unlocked state, the servo returns to 0° (locked). LCD returns to "Enter Password" prompt. System is ready for the next entry.

Pressing * manually locks the door at any time. LCD shows "Door Locked". Servo returns to locked position immediately.

Keypad Control Logic

Key Mapping

The 4×4 keypad uses two special keys as action triggers, mirroring the two-button simplicity of the reference design.

#

Hash Key — Submit / Confirm

Enter Password

Submits the entered digits as the password attempt. Triggers comparison and either unlocks or increments the wrong-attempt counter.

Change Password

After entering the master PIN, pressing # confirms and saves the newly entered 6-digit password to EEPROM permanently.

*

Star Key — Lock / Clear

Locked State

Manually locks the door immediately by returning the servo to 0°. Useful when leaving before the 10-second auto-relock timer expires.

Input State

Clears the current input buffer and resets the digit count to zero, allowing the user to re-enter the password from scratch.

Arduino Code

Core Sketch

The complete Arduino sketch — password comparison, servo control, EEPROM storage, and LCD feedback all in one file.

digital_lock.ino 
// ─────────────────────────────────────────────
//  Digital Lock System — Arduino UNO
//  Components: 4x4 Keypad, 16x2 LCD (I2C),
//              SG90 Servo, Buzzer, 2x LEDs
// ─────────────────────────────────────────────

#include <Keypad.h>
#include <LiquidCrystal_I2C.h>
#include <Servo.h>
#include <EEPROM.h>

// ── PIN DEFINITIONS ──────────────────────────
#define SERVO_PIN    9
#define BUZZER_PIN  8
#define LED_GREEN   7
#define LED_RED     6
#define PASS_LEN    6

// ── KEYPAD LAYOUT ────────────────────────────
const byte ROWS = 4, COLS = 4;
char keys[ROWS][COLS] = {
  {'1','2','3','A'},
  {'4','5','6','B'},
  {'7','8','9','C'},
  {'*','0','#','D'}
};
byte rowPins[ROWS] = {A0,A1,A2,A3};
byte colPins[COLS] = {A4,A5,2,3};

Keypad            keypad  = Keypad(makeKeymap(keys), rowPins, colPins, ROWS, COLS);
LiquidCrystal_I2C lcd     = LiquidCrystal_I2C(0x27, 16, 2);
Servo             servo;

// ── STATE ─────────────────────────────────────
char  enteredPass[PASS_LEN + 1];
char  storedPass[PASS_LEN + 1];
byte  digitCount   = 0;
byte  wrongAttempts = 0;
bool  isLocked     = true;

// ── SETUP ─────────────────────────────────────
void setup() {
  lcd.init(); lcd.backlight();
  servo.attach(SERVO_PIN);
  pinMode(BUZZER_PIN, OUTPUT);
  pinMode(LED_GREEN, OUTPUT);
  pinMode(LED_RED, OUTPUT);

  // Load password from EEPROM (or set default)
  for (byte i = 0; i < PASS_LEN; i++) {
    storedPass[i] = EEPROM.read(i);
    if (storedPass[i] < '0' || storedPass[i] > '9')
      storedPass[i] = (char)(i + 49); // default: 123456
  }
  storedPass[PASS_LEN] = '\0';

  lockDoor();
  lcd.print("  Arduino Lock  ");
  lcd.setCursor(0, 1);
  lcd.print("  Project v1.0  ");
  delay(2500);
  promptPassword();
}

// ── MAIN LOOP ─────────────────────────────────
void loop() {
  char key = keypad.getKey();
  if (!key) return;

  tone(BUZZER_PIN, 1200, 60);

  if (key == '*') {          // Clear / manual lock
    digitCount = 0;
    if (!isLocked) lockDoor();
    promptPassword();
    return;
  }

  if (key == '#') {          // Submit
    enteredPass[digitCount] = '\0';
    checkPassword();
    return;
  }

  if (digitCount < PASS_LEN) {
    enteredPass[digitCount++] = key;
    lcd.setCursor(digitCount - 1, 1);
    lcd.print('*');
  }
}

// ── HELPERS ───────────────────────────────────
void checkPassword() {
  if (strcmp(enteredPass, storedPass) == 0) {
    wrongAttempts = 0;
    grantAccess();
  } else {
    wrongAttempts++;
    if (wrongAttempts >= 3) lockout();
    else denyAccess();
  }
  digitCount = 0;
}

void grantAccess() {
  digitalWrite(LED_GREEN, HIGH);
  lcd.clear(); lcd.print("Access Granted!");
  lcd.setCursor(0, 1); lcd.print("Door Open");
  servo.write(90); isLocked = false;
  delay(10000); // 10-sec auto relock
  lockDoor(); promptPassword();
  digitalWrite(LED_GREEN, LOW);
}

void denyAccess() {
  digitalWrite(LED_RED, HIGH);
  lcd.clear(); lcd.print("Wrong Password!");
  lcd.setCursor(0, 1); lcd.print("Attempts: "); lcd.print(wrongAttempts);
  tone(BUZZER_PIN, 300, 800);
  delay(2000); digitalWrite(LED_RED, LOW);
  promptPassword();
}

void lockout() {
  lcd.clear(); lcd.print("!! LOCKED OUT !!");
  for (int i = 0; i < 5; i++) {
    tone(BUZZER_PIN, 200, 400); delay(500);
  }
  wrongAttempts = 0; promptPassword();
}

void lockDoor() {
  servo.write(0); isLocked = true;
}

void promptPassword() {
  lcd.clear(); lcd.setCursor(0, 0);
  lcd.print("Enter Password:");
  lcd.setCursor(0, 1); lcd.print("                ");
  lcd.setCursor(0, 1);
}

Conclusion

Project Summary

This project demonstrates a complete, working digital lock system using an Arduino UNO and readily available components. Starting from basic keypad interfacing, the system evolved into a full-featured security device with auto-locking, a 3-strike lockout, EEPROM-persistent password storage, visual and audio feedback, and a live Wokwi simulation for demonstration without physical hardware. The structured development timeline — from raw I/O testing in February to polished documentation in June — reflects a professional approach to embedded systems design. The project is well-positioned for real-world deployment and future expansion with Wi-Fi, biometrics, or RFID access.
